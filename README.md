# DocFeed

Feed your AI the freshest docs.

AI coding assistants like Cursor and Claude are powerful —
but they're trained on outdated data. Ask them to write
Vue 3.4 code and they'll give you Vue 2 patterns.

DocFeed solves this. Select your framework and version,
get a formatted context snippet, paste it into your AI tool.
Now your AI codes with today's docs, not last year's.

Currently supports: Vue 3 · Vue Router 4 · Pinia
